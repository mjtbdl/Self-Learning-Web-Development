    What is CSS

    CSS is the language we use to style a Web page. Before CSS programmers used to style their website with html. If you look at mid 90s websites you can understand the difference between sites styled with htmt. But today we can style better and easier with css. To do it we need another file inside our html file folder. Than if you link your file in heading area in your html file, you can start styling.

    CSS Types

    Inline, internal and external css are the types of css embeding.

    Inline

    In inline css you need to define the style of the tag in <> area.

    Internal

    In internal css you need a style tag in the header area.For example if you change the style of h1 color to the red. It will effect all h1 tags on that file. But it cant change other file's h1s.

    External

    In external css you need to create a css file. Every tag that you style in this file for example h1, will effect every h1 tags in every file.
